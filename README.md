# Gorące pytania

Prosta strona do losowania gorących pytań dla członków Toastmasters.

## Jak to dziala

- Strona wyswietla jedno losowe pytanie z pliku `pytania.txt`.
- Kazde pytanie powinno byc zapisane w osobnej linii.
- Repozytorium jest publiczne, wiec inne osoby moga proponowac poprawki i dodawac nowe pytania.

## Jak dodac nowe pytanie bez wiedzy technicznej

1. Otworz repozytorium na GitHub: [toastmasterspoznan/table-topics](https://github.com/toastmasterspoznan/table-topics)
2. Kliknij plik `pytania.txt`.
3. Wybierz opcje edycji pliku.
4. Dopisz nowe pytanie w nowej linii na koncu pliku.
5. Zapisz zmiane i utworz propozycje zmiany na GitHub.

## Jak zapisac pytania poprawnie

- Jedno pytanie = jedna linia.
- Nie numeruj pytan.
- Nie dodawaj pustych linii pomiedzy pytaniami.
- Mozesz pisac krotkie albo dluzsze pytania, byle byly zrozumiale.

## Jak przejrzec pytania na stronie

- Na stronie kliknij przycisk `i` w prawym gornym rogu.
- Znajdziesz tam link do podgladu pliku `pytania.txt` oraz link do repozytorium.

## Uruchomienie strony

Najprosciej opublikowac te strone przez GitHub Pages albo uruchomic ja przez prosty serwer HTTP. Bez tego przegladarka moze zablokowac wczytanie pliku `pytania.txt` przy otwieraniu strony bezposrednio z dysku.
